Requirements

  • PDO Extension
  • Allow_url_fopen enabled

Installation & Configuration

Installation

Update

If you already purchasing version 1.4 you can just in one click update to version 1.4.1
Backup your database and replace these files:
1 - File application/app/Http/Controllers/PaymentsController.php
2 - File application/app/Http/Controllers/Admin/PlayerSettings.php
3 - File application/resources/views/watch.blade.php
4 - File application/resources/views/upload.blade.php
5 - File content/assets/css/html5player.css
6 - Please follow how to create an Youtube API Key below (very important)

PayPal Configuration

Now to active paid subscription you have to config your paypal
1 - Go here : https://developer.paypal.com/developer/applications
2 - scroll down to 'REST API apps' and click 'Create App'
3 - Insert name and account then click Create App. Now you will get a client id code and secret code
4 - Open ('application/config/services.php') and edit paypal block with your info

Youtube API

If you want to upload videos from youtube you have to get an API key:
1 - Follow this lesson here to get an API key: https://www.slickremix.com/docs/get-api-key-for-youtube/
2 - When you get it open "application/config/youtube.php" and insert your api.

Translate iWatch

To translate our php script create new folder in application/resources/lang and give it a name like "es", "ar", "fr" ...
copy all files in application/resources/lang/en and open all files then translate
Now go to application/config/app.php and edit line 68 'locale' => 'en'
